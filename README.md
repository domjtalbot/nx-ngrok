## Installing

Using [npm](https://npmjs.com):

```bash
npm install -D nx-ngrok
```

Using [pnpm](http://pnpm.io):

```bash
pnpm add -D nx-ngrok
```

Using [yarn](http://yarnpkg.com):

```bash
yarn add -D nx-ngrok
```
